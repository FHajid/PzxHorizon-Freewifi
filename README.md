# 🌐 PZX Horizon - Portal Login Wi-Fi

Aplikasi Web Portal Login Wi-Fi berbasis React yang dirancang khusus untuk kebutuhan konektivitas area pesisir (**Coastal Connectivity Specialists**). Proyek ini memiliki manajemen jalur akses terpisah untuk Kru kapal (*Crew*) dan Tamu (*Guest*).

---

## 🚀 Cara Menjalankan Proyek (Local Development)

Ikuti langkah-langkah di bawah ini untuk menjalankan proyek di komputer lokal Anda setelah melakukan proses `git clone`:

```bash
# 1. Masuk ke dalam folder proyek
cd 02.pzx-login

# 2. Install semua dependensi/library yang dibutuhkan
npm install --legacy-peer-deps

# 3. Jalankan aplikasi secara lokal
npm start


 #untuk ke halaman Crew 
 - hanya taru /crew di depan Home(PzxHorizon-Freewifi) directory
# Untuk ke Guest itu ada Home directorynya tidak perlu di pindahkan
# Untuk Succes Login hanya di setelah di click dan keluar
